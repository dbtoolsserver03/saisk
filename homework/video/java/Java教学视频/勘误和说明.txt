Java视频勘误和说明

1、说明：在Lesson7中，CharsetTest.java文件中，通过修改Java系统属性pps.put("file.encoding","ISO-8859-1");的方式在J2SDK1.4.1下有效，但在J2SDK1.4.2下，修改这个系统属性不起作用了，大家下来测试的时候要注意，在学习Java的时候，有时候版本的差异会导致一些结果的改变。

2、在Lesson5中讲线程死锁的时候，我们看到运行时打印出一句话“sell():Thread-0 sell tickets:100”，在视频中，我说是先执行同步块中的代码，这是错误的，因为初始b为false，所以先执行了sell中的代码，导致打印出一句话，之后再进入同步块中的代码，这个时候就和视频中所说的死锁过程一致了。
